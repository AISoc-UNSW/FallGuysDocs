# Welcome

**FallGuys** is AISoc's Unity ML-Agents reinforcement learning course project: you train an AI agent to conquer a series of increasingly difficult Fall Guys–style obstacle courses, starting from simple movement and working up to a competitive self-play soccer level. Each level adds new mechanics, observations, and rewards for you to reason about — the point isn't to finish fastest, but to actually understand how the agent is learning.

**[github.com/AISoc-UNSW/FallGuys](https://github.com/AISoc-UNSW/FallGuys)** — the project repo. Clone it, follow the [installation guide](installation/macos.md), and you're ready to start training.

## Where to go from here

- **New to reinforcement learning or ML-Agents?** Start with the [Introduction](introduction/rl-theory.md) section.
- **Setting up your machine?** See [Installation](installation/macos.md).
- **Ready to train?** Head to [Levels](levels/level-0.md) and work through them in order.
